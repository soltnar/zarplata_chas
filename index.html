Зарплата сотрудников по сменам (месяц)

Версия:

Смена: больше 7 часов = 1, до 7 часов = 0.5. Сутки считаются с 04:00 до 03:59.

Как подготовить файлы в Saby

Загружайте файлы за один и тот же месяц. Для удобства используйте поле «Загрузить все файлы сразу (автоопределение)».

  1. Файл проходной: перейдите в Графики работ, откройте Отчеты → Пришел/ушел, задайте период и нажмите Выгрузить (Excel).
  2. Список сотрудников: откройте Сотрудники, в фильтре выберите сотрудников, работающих в нужном месяце (чтобы корректно определился ресторан при перемещениях). Перед выгрузкой уберите дублирующиеся группы (например, «Сдача отчетности» и сотрудники сторонних организаций), затем: Выгрузить → Список сотрудников в Excel → Основная информация.
  3. Файл начислений: откройте Зарплата, выберите список ресторанов и нажмите Выгрузить (Excel).
  4. Файлы премий: откройте Доп. начисления, выберите месяц и вид начисления Премия, отметьте нужные документы, затем Распечатать → Приказ Т-11 и сохраните PDF.

Совет: если выгружаете несколько PDF с премиями, загружайте их все одновременно — сервис объединит суммы по ФИО автоматически.

Загрузите файл проходной.
